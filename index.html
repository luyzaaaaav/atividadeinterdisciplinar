Comandos Linux e Comandos Git

O que é o Git?

Git é um sistema de controle de versão gratuito, de código aberto e distribuído, que rastreia as alterações em projetos de software, permitindo que múltiplos desenvolvedores colaborem de forma eficiente

O que é o GitHub?

O GitHub é uma plataforma na nuvem que utiliza o Git para hospedar projetos de desenvolvimento, permitindo que desenvolvedores armazenem, gerenciem e colaborem no código-fonte de forma eficiente, rastreando alterações, revisando o trabalho de colegas e contribuindo para projetos de código aberto. Essencialmente, funciona como um "social code host" que oferece controle de versão, gestão de projetos e ferramentas para trabalho em equipa num único local, facilitando a criação e partilha de software.

Quais são suas diferenças?

1 - Git: ferramenta local (linha de comando ou GUI), controla versão no seu computador.
2 - GitHub: serviço remoto (web) que armazena repositórios Git e adiciona recursos sociais/colaborativos.
3 - Você pode usar Git sem GitHub; não dá pra usar GitHub sem Git.

Como eu crio um Repositório no GitHub?

1 - Faça login no site do GitHub
2 - Clique no botão "New repository".
3 - Preencha o nome do repositório, descrição, e escolha se ele será público ou privado.

Como eu ligo um projeto existente no GitHub no meu PC?

git commit -m "commit"
# adicionar o repositório remoto (substitua o link pelo que foi copiado do GitHub)
git remote add origin (link copiado)
git push -u origin main
