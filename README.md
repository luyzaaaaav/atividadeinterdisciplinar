Este projeto interdisciplinar propõe a criação de um site estático sobre Economia Doméstica e Finanças Pessoais, integrando conteúdos de Matemática, Geografia, Sociologia e Português. O objetivo é promover a educação financeira, incentivando o consumo consciente e a organização das finanças. O site deve incluir uma página inicial de apresentação, um simulador de orçamento mensal com cálculos automáticos, dicas práticas de consumo, infográficos econômicos e um glossário de termos financeiros. A proposta estimula o uso de HTML, CSS e JavaScript de forma criativa e educativa, unindo conhecimentos técnicos e sociais. O resultado final será publicado no Netlify até o dia 29 de outubro de 2025.
